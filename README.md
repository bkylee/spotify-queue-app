Visit https://spotify-queue-neon-fox.azurewebsites.net/ to see the app running. 

I created this app to allow users who don't have spotify or premium to have a our own Spotify Jam using my personal account. Users can access this website and do the following: 
- See what I'm currenlty playing
- See the next 10 songs in my queue
- Search and add songs to the queue

I used Claude AI for the app creation and Azure Web App for hosting.
